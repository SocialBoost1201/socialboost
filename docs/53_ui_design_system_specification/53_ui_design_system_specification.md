# SocialBoost 公式Webサイト UIデザインシステム設計書

---

## 文書目的

本書は SocialBoost 公式WebサイトのUIデザインルール、デザインコンポーネント、カラーパレット、タイポグラフィ、余白ルールなどを定義する。

UIデザインシステムを定義することで、
- デザインの一貫性
- 開発効率の向上
- 保守性の向上
- 拡張性の確保

を実現する。

本書は以下の設計書と連携する。
- UX設計書
- 画面設計書
- コンポーネント設計書
- モーション設計書

## カラーパレット設計

### カラー定義（ブランドカラー基準）

| 用途 | トークン名 | 説明 |
|:---|:---|:---|
| **メインカラー** | `Brand Primary` | ロゴの色を基準に設定 |
| **アクセントカラー** | `Brand Accent` | CTA等の強調に使用 |
| **背景（ライト）** | `Background Light` | 白系の標準背景 |
| **背景（ダーク）** | `Background Dark` | ダークセクション背景 |
| **テキスト（主）** | `Text Primary` | 本文テキストの主色 |
| **テキスト（副）** | `Text Secondary` | 補助テキスト・キャプション |

### カラー設計ルール
- CTAにはアクセントカラーを使用する
- 背景は白を基本とする
- 可読性を担保する強いコントラストを維持する

## タイポグラフィ設計

### フォント
- **フォントソース:** Google Fonts
- **見出しフォント:** Sans Serif（モダンで読みやすいゴシック系）
- **本文フォント:** Readable Sans Serif（視認性重視）

### 文字サイズ設計

| レベル | 用途 |
|:---|:---|
| **H1** | ヒーロータイトル |
| **H2** | セクションタイトル |
| **H3** | カードタイトル、サブセクション |
| **Body** | 本文 |

---

## 余白（スペーシング）設計

### 基本グリッド
**8pxグリッドシステム** を採用する。

### 余白ルール

| 余白種別 | サイズ感 |
|:---|:---|
| セクション間余白 | 大（広めの余白で視覚的に区切る） |
| カード間余白 | 中 |
| 要素間余白 | 小 |

---

## UIコンポーネント設計

### ボタンデザイン

| 種別 | 用途 | スタイル |
|:---|:---|:---|
| **Primary Button** | メインCTA | アクセントカラー / 角丸 / ホバーエフェクト / 軽いシャドウ |
| **Secondary Button** | サブCTA | アウトライン系 / 控えめなホバー |

### カードデザイン
- **用途:** サービス紹介、実績一覧
- **構成要素:** 画像、タイトル、説明文
- **スタイル:** 角丸、軽いシャドウ、ホバーエフェクト

### アイコン
- **用途:** サービス説明等
- **スタイル:** シンプルなラインアイコン

### 画像ルール
- 高解像度画像を使用する
- `Next Image` で最適化配信する

---

## レスポンシブUIとアクセシビリティ

### レスポンシブUI

| デバイス | レイアウト方針 |
|:---|:---|
| **PC** | 広い余白を活用した複数カラムレイアウト |
| **タブレット** | 2カラム中心 |
| **スマートフォン** | 1カラム中心 |

### アクセシビリティ
- 十分なコントラスト比の確保
- 読みやすいフォントサイズの設定

### UI一貫性ルール
- 同じ要素は同じデザインを使う
- CTAボタンのスタイルはサイト全体で統一する
- カードデザインはサービスと実績で統一する
