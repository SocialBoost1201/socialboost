# SocialBoost 公式Webサイト コンテンツ設計書

---

## 文書目的

本書は SocialBoost 公式Webサイトにおけるコンテンツ構造、コピー設計、情報配置ルールを定義する。

コンテンツ設計は単なる文章作成ではなく、
- ユーザー理解
- 信頼形成
- SEO最適化
- コンバージョン誘導

を同時に成立させるための設計である。

本書ではページ別コンテンツ構造、コピー設計、情報配置、CTA設計を定義する。

本書は以下の設計書と連携する。
- UX設計書
- SEO設計書
- 画面設計書
- マーケティング設計書
- 情報設計書

## コンテンツ設計の基本方針

1. 短く理解しやすい
2. 専門性は出すが難しくしない
3. 営業臭を出しすぎない
4. 信頼感を重視する
5. 実績を中心に信頼形成する

### コンテンツの4つの役割

| 役割 | 説明 |
|:---|:---|
| **理解** | 何をしている事業かを理解させる |
| **信頼** | 実績や専門性を提示する |
| **比較** | サービスの違いを理解させる |
| **行動** | 問い合わせへ誘導する |

## ページ別コンテンツ構造

### トップページ
| セクション | コンテンツ |
|:---|:---|
| **ヒーロー** | キャッチコピー / サブコピー |
| **サービス概要** | 提供サービス一覧（カード） |
| **実績紹介** | 制作実績カード |
| **選ばれる理由** | 強み・特長 |
| **CTA** | お問い合わせ誘導 |

#### ヒーローコピー設計
- **目的:** サイトの内容を瞬時に理解させる
- **キャッチコピー例:** 「Web制作・システム開発・AI導入を支援するデジタル戦略パートナー」
- **サブコピー例:** 「売上に直結するWeb設計、AI活用、システム開発まで一貫支援」

### サービスページ
- サービス概要 → 各サービス説明 → 導入メリット → CTA

**サービス内容:** Web制作 / LP制作 / Webシステム開発 / アプリ開発 / AI導入支援

### 料金ページ
- 料金概要 → 料金プラン → 補足説明 → CTA

**料金説明方針:** 価格だけでなく価値を説明する。安売り印象を避ける。

### 実績一覧ページ
- 実績カード一覧（画像、タイトル、概要）

### 実績詳細ページ
- プロジェクト概要 → 課題 → 実施内容 → 成果 → 画像またはPDF

### お問い合わせページ
- 説明文（相談歓迎の文脈） → 問い合わせフォーム → 送信ボタン
