# SocialBoost 公式Webサイト Analytics・計測設計書

---

## 文書目的

本書は SocialBoost 公式Webサイトにおけるアクセス解析、ユーザー行動分析、コンバージョン測定の設計を定義する。

本サイトは単なる情報サイトではなく、
- 問い合わせ獲得
- 営業ツール
- 信頼形成

を目的としているため、ユーザー行動を正しく測定し改善サイクルを回す必要がある。

本書ではアクセス解析、イベント計測、コンバージョン計測、ユーザー行動分析を定義する。

本書は以下の設計書と連携する。
- UX設計書
- SEO設計書
- マーケティング設計書
- パフォーマンス設計書
- 運用設計書

## 計測目的と主要KPI

### 計測目的
- 訪問者数の把握
- ユーザー導線の把握
- コンバージョン率の改善
- 離脱ポイントの特定

### 目標KPI

| 指標 | 目標値 |
|:---|:---|
| **月間訪問数** | 1,000 |
| **月間問い合わせ数** | 10 |
| **CVR（コンバージョン率）** | 1% |
| **実績ページ閲覧数** | KPIとして追跡 |

---

## イベント計測設計

### 計測イベント一覧

| イベント名 | 内容 |
|:---|:---|
| `page_view` | ページ閲覧 |
| `contact_view` | お問い合わせページ閲覧 |
| `contact_submit` | 問い合わせ送信 |
| `service_view` | サービスページ閲覧 |
| `works_view` | 実績一覧ページ閲覧 |
| `works_detail_view` | 実績詳細ページ閲覧 |

### カスタムイベント設計

| ユーザー操作 | イベント名 |
|:---|:---|
| 問い合わせボタンクリック | `contact_click` |
| 問い合わせ送信完了 | `contact_submit` |
| 実績カードクリック | `works_click` |
| CTAボタンクリック | `cta_click` |

---

## コンバージョン定義

### 主要CV（Primary Conversion）
- **問い合わせ送信**（`contact_submit`）

### 副次CV（Micro Conversion）
- お問い合わせページ閲覧（`contact_view`）
- サービス詳細閲覧（`service_view`）
- 実績詳細閲覧（`works_detail_view`）

## ユーザー行動分析・ヒートマップ・SEO分析

### ユーザー行動分析（GA4）
- 流入経路
- 人気ページ
- 離脱ページ
- 滞在時間

### ヒートマップ分析（Microsoft Clarity）
- クリック分布
- スクロール深度
- 離脱ポイント

### SEO分析（Search Console）
- 検索クエリ
- 検索順位
- クリック率（CTR）
