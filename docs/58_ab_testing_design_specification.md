# SocialBoost 公式Webサイト A/Bテスト設計書

---

## 文書目的

本書は SocialBoost 公式Webサイトにおけるコンバージョン率改善を目的としたA/Bテストの方針、対象要素、実施方法を定義する。

本サイトは問い合わせ獲得型サイトであるため、デザイン・コピー・CTA配置などを継続的に改善する必要がある。

A/Bテストにより以下を実現する。
- CVR向上
- 離脱率低下
- ユーザー理解の改善
- UX最適化

本書ではテスト対象、テスト方法、評価指標、改善プロセスを定義する。

本書は以下の設計書と連携する。
- UX設計書
- Analytics設計書
- マーケティング設計書
- コンテンツ設計書

## A/Bテストの基本方針

1. 小さく改善する
2. 仮説ベースで実施する
3. データで判断する
4. ユーザー体験を損なわない

---

## A/Bテスト対象と具体例

### 1. ヒーローコピー テスト
- **目的:** サイトの理解度向上
- **パターンA:** 「Web制作・システム開発・AI導入を支援するデジタル戦略パートナー」
- **パターンB:** 「売上を伸ばすWebサイトとAI活用をワンストップで支援」

### 2. CTAボタン テスト
- **対象:** トップページCTA
- **パターンA:** 「お問い合わせ」
- **パターンB:** 「無料相談してみる」

### 3. 実績表示 テスト
- **対象:** 実績一覧ページ
- **パターンA:** カード一覧のみ
- **パターンB:** カード＋カテゴリフィルター付き

### 4. 問い合わせ導線 テスト
- **対象:** ページ下部CTA
- **パターンA:** 「お問い合わせ」
- **パターンB:** 「相談してみる」

### 5. 料金説明 テスト
- **目的:** 価格不安の低減
- **パターンA:** 料金一覧のみ
- **パターンB:** 料金＋考え方の説明付き

---

## 評価指標とテスト方法

### 評価指標
- **主要指標:** 問い合わせ送信率（CVR）
- **副次指標:** お問い合わせページ到達率、CTAクリック率、ページ滞在時間

### テスト方法
| 項目 | 設定 |
|:---|:---|
| **トラフィック分割** | 50% / 50% |
| **テスト期間** | 最低2週間 |
| **統計的有意性** | 95%以上 |

### テスト優先順位

| 優先度 | テスト対象 |
|:---|:---|
| 1 | ヒーローコピー |
| 2 | CTAボタン |
| 3 | 料金ページ |
| 4 | 実績表示方法 |
